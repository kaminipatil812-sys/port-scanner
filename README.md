# port-scanner
A simple Python-based TCP Port Scanner that uses socket programming to identify open ports on a target host. Built for educational purposes to demonstrate basic networking, cybersecurity concepts, and Python socket programming. Intended for use only on systems you own or have permission to test.

# Project Purpose
The purpose of this project is to learn the fundamentals of network security and Python socket programming by developing a simple TCP port scanner. It helps users understand how open ports are identified and demonstrates basic network scanning techniques for educational and authorized security testing.

# Desing Tools
Python 3 – Core programming language
Socket Module – TCP network communication and port scanning
VS Code (or any Python IDE) – Code development
Git & GitHub – Version control and project hosting
